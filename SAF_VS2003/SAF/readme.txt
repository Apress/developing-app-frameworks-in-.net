The source code is categorized into two groups: SAF and Testing projects

Folders that start with "SAF" contain the source code for the various SAF components
SAF.sln file on the root folder represents the solution containing the only SAF source code.

Folders that start with "Test" contain the testing project for the a specific SAF component
Each "Test.*" folder contains VS.NET solution file for the testing project

Please refer to "Testing project" section in each book chapter for more information and instruction on how to run the demo in the Testing project
